teknologi/Programmer:


dbeaver
dbeaver er database management værktøj til database udviklere og administratorer. som har en grafisk brugerflade til finde rundt med.






Node.JS
Node.JS is a server-side-programming som har nogle API funktioner. kan lave begivenheds styringsprogrammer. og i alt gør udvikling nemmere og vedligeholdelse hurtigt og nem.  




Express.JS
Express.JS  er et node.js layout hvor du kan arbejde med url, brugerlogin, session og håndtering af klient anmodningen.


MySQL
MySQL er database management system hvor du kan client computer for adgang til data ved hjælp software der hedder RDBMS.




CSS
CSS er et sprog der bruges til gøre andre hjemmesider så blive pænere. Hvor man kan om på farver,layout og skrifttype, placering.


Bootstrap
Bootstrap er et sprog hvor forarbejdede layout som er bygget op af JavaScript, CSS og html. Du kan spare meget tid ved at bruge fordi slipper for skulle lave lange code for det layout.




JavaScript
JavaScript bruge man til at lave interaktiv på hjemmesiden, animation, algoritme.
